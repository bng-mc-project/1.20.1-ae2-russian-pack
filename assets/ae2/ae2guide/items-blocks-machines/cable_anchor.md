# Тросовый анкер

<GameScene zoom="6" background="transparent">
  <ImportStructure src="../assets/assemblies/cable_anchor.snbt" />
  <IsometricCamera yaw="195" pitch="30" />
</GameScene>


Маленькие декоративные шипы на кабелях, которые предотвращают соединение кабелей, создают лестницы с помощью кабелей или не позволяют кабелям соединяться к стенам вокруг них. Также используется для изготовления <ItemLink id="facade" />.

Тросовые анкеры предотвращают образование соединений со стороны на которой они установлены.

Вы можете карабкаться по ним, как по лестницам.
