# Крохотный динамит

<BlockImage id="tiny_tnt" scale="8" />

Маленький динамит для маленьких взрывов. Полезен для изготовления <ItemLink id="quantum_entangled_singularity" />.

Есть возможность выключить уничтожение блоков взрывом, что позволит создавать сингулярность без возможности гриферства, если вы захотите выключить динамит и криперов на вашем сервере.
